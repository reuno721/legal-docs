Plimo 계정 및 데이터 삭제 안내

최종 업데이트: 2026년 5월 9일

이 페이지는 Plimo 사용자가 계정 연결 및 관련 데이터를 삭제하는 방법을 안내합니다. Plimo는 개인 개발자가 제공하는 식물 물주기 알림 및 화분 관리 앱입니다.

1. 앱에서 직접 삭제하는 방법

  1. Plimo 앱을 엽니다.
  2. 홈 화면 오른쪽 위의 설정 버튼을 누릅니다.
  3. 계정·앱 탭을 엽니다.
  4. 데이터 관리 섹션으로 이동합니다.
  5. 계정 및 모든 데이터 삭제를 선택합니다.
  6. 확인 대화상자에서 삭제를 확정합니다.

3. 삭제되는 데이터

  • Plimo의 Google 계정 연결 정보
  • Cloud Firestore에 백업된 식물 정보, 관리 기록, 통계, 안내 프로필
  • 이 기기에 저장된 식물 정보, 관리 기록, 설정
  • 이 기기의 앱 내부 저장소에 저장된 식물 대표 사진

4. 삭제되지 않는 데이터

  • 사용자의 Google 계정 자체는 삭제되지 않습니다.
  • 사용자가 직접 기기 갤러리나 다른 앱에 보관 중인 원본 사진은 삭제되지 않습니다.
  • 법적 의무, 보안, 분쟁 해결을 위해 필요한 일부 기록은 제한된 기간 동안 보관될 수 있습니다.

5. 일부 데이터만 삭제하는 방법

계정을 삭제하지 않고 일부 데이터만 삭제할 수도 있습니다.

  • 식물 또는 기록 삭제: 앱에서 해당 식물 또는 기록을 삭제합니다.
  • 클라우드 백업 삭제: 설정 > 계정·앱 > 데이터 관리에서 클라우드 백업 삭제를 선택합니다.
  • 이 기기 기록 초기화: 설정 > 계정·앱 > 데이터 관리에서 데이터 초기화를 선택합니다.

6. 처리 기간

앱에서 직접 삭제한 데이터는 즉시 삭제 절차가 시작됩니다. 이메일 요청은 본인 확인 및 요청 확인 후 합리적인 기간 내에 처리합니다.
